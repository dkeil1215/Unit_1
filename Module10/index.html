How to Use This Dashboard

Use the dropdown menus to choose a map mode, variable, year, and month.

Click a county on the map or bar chart to select it. Use Compare County to compare two counties in the trend chart.

Single Variable mode maps one variable. Bivariate mode maps two variables together using the color grid legend.
